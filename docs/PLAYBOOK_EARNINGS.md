# Earnings Playbook

This is the "make the play without spiraling" guide.

## Mission

Use the dashboard to decide whether a name deserves:

- `Make A Move`
- `Stalk It`
- `Wait`

The goal is not to trade more. The goal is to trade only when the desk agrees with itself.

## What Counts As A Real Earnings Play

A real play is one where these things line up at the same time:

1. The ticker passes the `Trade Gate Sanctum`.
2. The setup is not blocked.
3. The catalyst window is close enough to matter.
4. The signal is live or very close.
5. The score stack is not being carried by one fake number.

If those are not true together, the answer is almost always `Wait`.

## Step By Step

### 1. Read the board, not your feelings

Start with:

- `Top Opportunities`
- `Trade Gate Sanctum`
- `Chosen Vessel`

If the name is not showing up in those three places with a coherent story, it is probably not the one.

### 2. Respect the gates first

Before you even think about structure, check:

1. `Readiness`
2. `Confidence`
3. `Days until earnings`
4. `Trigger state`
5. `Setup bias`

The gate exists to keep you from forcing low-quality action.

### 3. Decide the posture

Use this language on purpose:

- `Make A Move`
  - trigger is live, timing is clean, and the score stack agrees
- `Stalk It`
  - real edge exists, but entry still wants patience
- `Wait`
  - interesting name, bad moment

If you cannot say which posture it is in one sentence, you are not ready to trade it.

### 4. Pick one thesis only

Every trade should have one sentence:

- "I am playing the earnings window because the trigger is live and the setup is aligned."
- "I am stalking this because the thesis is good but the timing is not ready yet."

If you need three paragraphs to justify it, the desk is probably telling you no.

### 5. Build the order off the desk

Use:

- primary route
- secondary route
- readiness
- timing
- trigger

Do not freestyle structure once the desk already chose one.

### 6. Size like you can be wrong

Rules:

- size smaller when confidence is just barely passing
- size smaller when the setup is more complex
- size smaller when the signal only recently woke up
- never let one earnings name become a referendum on your intelligence

### 7. Journal the trade immediately

The desk should always be able to answer:

- why this name
- why now
- what invalidates it
- what the fallback route was

If you do not capture that before the trade, you are turning learning into theater.

## No-Overthinking Rules

1. If the name fails the gate, do not debate it.
2. If the trigger is dead, it is not a `Make A Move`.
3. If the setup says `Avoid`, do not invent hero logic.
4. If you missed the clean entry, do not chase the candle to feel included.
5. If two names look close, take the one the desk ranks higher.
6. If the thesis depends on "this time it feels different," stand down.

## End Of Day Review

After the market:

1. Which names passed the gates?
2. Which names were real opportunities?
3. Which names looked loud but were fake?
4. Did the desk save you from anything stupid?
5. Did you override it anywhere?

That review matters more than one lucky winner.
