# Long-Term Accumulation Playbook

This is the lane for names you already believe in and want to buy better, not faster.

## Mission

The long-term desk is for:

- conviction names
- calmer entries
- buying quality while the heat is lower

It is not for random "looks cheap" fishing.

## What The Accumulation Lane Means

The `Accumulation Vault` is trying to answer:

- which names still have real conviction
- which names are not already overheated
- which names look calmer than the short-term crowd realizes

Version one of this lane uses the desk's existing score stack to spot value plus calm tape. It is a buy-the-dip proxy, not a perfect institutional valuation model.

## Step By Step

### 1. Start with names you would want anyway

Only consider a long-term buy if you can answer yes to this:

- "Would I still want to own this in six to twelve months if it went nowhere for a while?"

If not, it does not belong in the accumulation lane.

### 2. Look for calm, not excitement

The best long-term adds usually have:

- strong value contribution
- lower heat
- less chase behavior
- enough business quality to deserve patience

This lane is supposed to feel boring compared with the earnings desk. That is a good sign.

### 3. Use the accumulation posture

The desk uses three postures:

- `Accumulate`
  - conviction is high and the name is calm enough to keep buying
- `Nibble`
  - worth starting or adding slowly, but not a rush
- `Wait For Weakness`
  - still a good company maybe, but not a good discount yet

### 4. Build in pieces

Rules:

1. never go all-in on the first red day
2. start small
3. add only if the thesis is the same and price gives you more room
4. stop adding if your reason changes from conviction to coping

### 5. Separate this from earnings emotion

Do not use the long-term lane to justify an earnings miss or a bad short-term trade.

Long-term buys should be their own decision, with their own timeline.

## No-Overthinking Rules

1. If you would not want the name six months from now, do not accumulate it.
2. If the business quality is unclear, the price discount does not matter.
3. If you are buying just because it is red, that is not conviction.
4. If the stock is ripping and you suddenly call it "long term," you are probably just chasing slower.
5. If the thesis changes, pause adds until the story is rebuilt.

## What To Track

When you add a long-term name, write down:

- why you already believed in it
- why the current price is better than before
- what would make you stop adding
- how many pieces you want to build the position in

## Next Evolution

The next stronger version of this lane should add:

- actual 3-month and 6-month drawdown
- distance from moving averages
- trend damage versus simple volatility compression

That will make the discount signal more literal. Right now the lane is a disciplined conviction proxy, which is still much better than impulse buying.
