# Authority Boundary

The unattended loop may refresh data, recompute analytics, create paper/shadow evidence, run evaluations, and write local knowledge notes.

It may not:

- submit a broker order;
- enable live trading;
- approve, reject, close, or promote an operator ticket;
- change risk policy;
- change the eligible trading universe;
- treat scenario observations as executable fills.

Any future self-modifying optimizer must operate in an isolated worktree, against immutable evaluators, and produce reviewable patches. It receives no live-book authority.
